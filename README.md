## About The Project

 React.js SPA that allows users to:

* Load gifs from Giphy API
* Search for a specific gif 



## Getting Started


### Installation

1. Download the files

2. Install NPM packages 
```sh
npm install
```
3. Run the web application
```sh
npm start
```
